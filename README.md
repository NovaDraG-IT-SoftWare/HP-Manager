# HP-Manager (Hyper Prime) by NovaDraG IT SoftWare
​HP-Manager — это мощный, легковесный и интуитивно понятный файловый менеджер для устройств на базе Android. Созданный разработчиками для разработчиков, он сочетает в себе высокую скорость работы файловой системы и современный интерфейс с глубокой кастомизацией.
​🌟 Почему HP-Manager?
​В отличие от громоздких аналогов, HP-Manager сфокусирован на чистой производительности. Мы убрали всё лишнее, оставив только то, что нужно для эффективного управления данными.
​💎 Ключевые особенности:
​📦 Интеллектуальный установщик APK: Встроенная логика считывания метаданных. Приложение показывает реальную иконку, название и пакет (Package Name) еще до установки, защищая вас от "котов в мешке".
​🔍 Мгновенный поиск: Фильтрация сотен файлов происходит в реальном времени благодаря оптимизированным алгоритмам обработки списков.
​🎨 Адаптивный интерфейс: Полная поддержка динамических тем (Light/Dark). Интерфейс подстраивается под ваши предпочтения, обеспечивая комфорт для глаз в любое время суток.
​⚡ Тактильная отдача (Haptic Feedback): Каждое важное действие сопровождается мягким виброоткликом, создавая ощущение взаимодействия с физическим объектом.
​📁 Работа с архивами: Быстрый просмотр содержимого ZIP-архивов без необходимости полной распаковки.
​🖼 Медиа-превью: Благодаря интеграции Glide, вы видите эскизы фотографий и изображений мгновенно, без задержек при скроллинге.
​🛠 Технический стек
​Проект реализован с использованием нативных технологий Android, что гарантирует минимальное потребление оперативной памяти и максимальную отзывчивость:
​Core: Java 8+ / Android SDK.
​Build System: Gradle.
​IDE: Разработано исключительно в AndroidIDE (Mobile Development Workflow).
​Architecture: UI-driven development с кастомными адаптерами для управления памятью.
